Liste des commandes:

!play - Suivi d'une url youtube, cette commande permet de lancer la vidéo dans le channel audio dans lequel vous êtes. 
!ysearch - Prend l'argument en paramètres pour effectuer une recherche youtube et lire la première vidéo trouvée.
!gay - Permet de lancer une vidéo youtube prédéfinie dans le channel audio dans lequel vous êtes.
!connerie - Permet de lancer une vidéo youtube prédéfinie dans le channel audio dans lequel vous êtes.
!defused - Permet de lancer une vidéo youtube prédéfinie dans le channel audio dans lequel vous êtes.
!planted - Permet de lancer une vidéo youtube prédéfinie dans le channel audio dans lequel vous êtes.
!stop - Coupe la lecture en cours.
!help - Affiche la liste des commandes disponibles.
